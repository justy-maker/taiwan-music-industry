# LiveHouse 展演生態研究

> 更新日期：2026-02-09
> 研究範圍：台灣 LiveHouse 產業生態系統

---

## 一、產業定義與規模

### 1.1 什麼是 LiveHouse？

LiveHouse 是指專門提供小型現場音樂演出的場地，通常具備以下特徵：
- 容納人數：50-1,500人
- 設有專業音響、燈光設備
- 提供酒水餐飲服務
- 以票務收入+酒水為主要營收

### 1.2 台灣 LiveHouse 產業規模

| 指標 | 數據 | 備註 |
|-----|-----|-----|
| 場地數量 | 約 60-80 間 | 全台估計 |
| 年演出場次 | 約 3,000-5,000 場 | 各場地合計 |
| 年產值估計 | 約 5-10 億元 | 含票務+酒水 |
| 從業人員 | 約 500-1,000 人 | 場地直接雇用 |

## 二、經營模式分析

### 2.1 收入結構

| 收入來源 | 佔比 | 說明 |
|---------|-----|-----|
| **票務收入** | 40-50% | 售票分潤或場租 |
| **酒水餐飲** | 30-40% | 主要利潤來源 |
| **場地租借** | 10-15% | 記者會、活動等 |
| **其他** | 5-10% | 周邊、寄售等 |

### 2.2 與樂團的合作模式

#### 模式一：場租制
- 樂團支付固定場租（$10,000-50,000）
- 票房歸樂團所有
- 適合：有粉絲基礎的樂團

#### 模式二：分潤制
- 場地提供場地、設備、人力
- 票房依比例分配（常見 7:3 或 6:4）
- 場地承擔較多風險

#### 模式三：保底+分潤
- 場地保證基本場次費
- 超過門檻後再分潤
- 平衡風險

### 2.3 成本結構

| 成本項目 | 佔營收比例 | 說明 |
|---------|----------|-----|
| 房租 | 20-35% | 最大固定成本 |
| 人事 | 20-30% | 音控、燈光、吧台 |
| 設備維護 | 5-10% | 音響、燈光保養 |
| 水電 | 5-8% | |
| 酒水成本 | 10-15% | 毛利約 50-70% |
| 行政雜支 | 5-10% | |

## 三、法規環境

### 3.1 相關法規

| 法規名稱 | 主管機關 | 影響 |
|---------|---------|-----|
| 建築法 | 內政部 | 場地使用分區、消防 |
| 消防法 | 內政部消防署 | 消防設備、逃生動線 |
| 菸害防制法 | 衛福部 | 室內禁菸 |
| 噪音管制法 | 環保署 | 音量限制 |
| 食品衛生法 | 衛福部 | 餐飲證照 |
| 營業稅法 | 財政部 | 娛樂稅、營業稅 |

### 3.2 營業登記類別

LiveHouse 常見登記類別：
1. **視聽歌唱業**（KTV條例管轄）
2. **餐飲業**（餐廳+表演）
3. **展演空間**（文化部輔導）
4. **複合式經營**

### 3.3 2020年《音樂展演空間業輔導辦法》

文化部2020年推出專門輔導辦法：
- 放寬建築使用類組認定
- 簡化消防審查流程
- 提供輔導金補助
- 建立專業證照制度

## 四、產業困境分析

### 4.1 主要困境

| 困境 | 嚴重程度 | 說明 |
|-----|---------|-----|
| **高房租** | ⭐⭐⭐⭐⭐ | 都市精華區租金壓力大 |
| **法規限制** | ⭐⭐⭐⭐ | 消防、噪音、使用分區 |
| **疫情衝擊** | ⭐⭐⭐⭐ | 2020-2022重創產業 |
| **客源不穩** | ⭐⭐⭐ | 依賴個別演出熱度 |
| **人才流失** | ⭐⭐⭐ | 低薪、工時長 |

### 4.2 地下社會事件（2012）

**事件背景**：
- 台北師大商圈知名 LiveHouse
- 因違反土地使用分區規定被迫歇業
- 引發獨立音樂圈強烈反彈

**影響**：
- 促成文化部正視 LiveHouse 法規問題
- 推動後續輔導辦法制定
- 成為產業發展轉折點

### 4.3 疫情影響（2020-2022）

| 階段 | 時間 | 影響 |
|-----|-----|-----|
| 第一波 | 2020.1-6 | 演出全面取消，營收歸零 |
| 復甦期 | 2020.7-2021.4 | 逐步恢復，人數限制 |
| 第二波 | 2021.5-8 | 三級警戒，再度停業 |
| 恢復期 | 2021.10-2022 | 逐步開放，但仍受限 |
| 全面開放 | 2022.10起 | 回復正常營運 |

**政府紓困**：
- 藝文紓困補助
- 薪資補貼
- 場地租金補助

## 五、成功案例分析

### 5.1 Legacy 系列

**成功因素**：
- 規模經濟（台北、台中、高雄三館）
- 專業團隊經營
- 穩定卡司來源
- 品牌效應

**經營數據**（估計）：
- Legacy Taipei：年演出 200+ 場
- 年營收估計：1-2 億元

### 5.2 女巫店

**成功因素**：
- 歷史品牌（1996年創立）
- 社群經營
- 低成本營運
- 忠實客群

**經營數據**（估計）：
- 年演出：150-200 場
- 小而美的永續經營

### 5.3 鐵花村

**成功因素**：
- 地方政府支持
- 原民音樂特色
- 結合觀光
- 社區營造

**經營數據**（估計）：
- 年演出：300+ 場
- 結合市集、文創

## 六、產業趨勢

### 6.1 新興趨勢

1. **線上線下整合**
   - 線上售票+直播
   - NFT門票嘗試
   
2. **複合式經營**
   - LiveHouse + 餐飲
   - LiveHouse + 共享空間
   
3. **政府場館增加**
   - 台北流行音樂中心
   - 高雄流行音樂中心
   - 對民間場地形成競爭

4. **ESG 永續經營**
   - 環保杯政策
   - 減少一次性用品

### 6.2 挑戰與機會

| 挑戰 | 機會 |
|-----|-----|
| 大型場館競爭 | 差異化小型演出 |
| 房租持續上漲 | 政府輔導計畫 |
| 疫情反覆風險 | 線上演出技術成熟 |
| 人才培育不足 | 產學合作機會 |

---

## 七、政策建議方向

1. **持續放寬法規**：簡化消防、使用分區審查
2. **租金補貼**：針對文化場域提供租金補助
3. **人才培育**：建立音響、燈光技術證照
4. **稅務優惠**：娛樂稅減免
5. **場地媒合**：閒置空間活化
